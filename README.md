# sales_data_analysis

Simple project for superstock_sales.csv file that included.


<h3>There is analysis for :</h3>

<ul>

  <li>Top 10 products by sales</li>
  <li>Most selling products</li>

  <li>Most profitable category and sub-category</li>
  
</ul>
